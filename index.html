About

Anyconversion.com is a measurement unit conversion website. Unlike other websites on the net, where you have to select a category to find the unit you want to convert, anyconversion.com let's you type and search the unit saving time and confusion, while automatically loading the relevant result units.

So, just start typing the unit you want to convert, enter a value, select the conversion unit and you are good to go.
